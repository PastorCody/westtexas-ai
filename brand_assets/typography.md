# Typography

## Headlines
- Font: Instrument Serif (Google Fonts)
- Fallback: Georgia, serif
- Weight: 400 (normal)
- Letter-spacing: -0.03em
- Hero size: clamp(3rem, 7vw, 5.5rem)
- Section size: clamp(2rem, 4vw, 3.25rem)

## Body / UI
- Font: DM Sans (Google Fonts)
- Fallback: system sans-serif
- Size: 16-18px
- Line-height: 1.65-1.7

## Never Use
Inter, Roboto, Arial, Space Grotesk
